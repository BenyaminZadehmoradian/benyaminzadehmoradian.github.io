# LaTeX Source Files

## Compilation
This resume uses XeLaTeX (for fontspec support).

```bash
xelatex resume_en.tex
xelatex resume_en.tex   # run twice for correct page labels
```

## Requirements
- XeLaTeX
- Liberation Serif font (or replace with Times New Roman if available)
- Packages: geometry, hyperref, enumitem, xcolor, fontspec, microtype

## Output
Compiled PDF → `../files/resume_en.pdf` (served on the website for download)
